## 📖 Project Description

**P.folio** is a premium, pixel-perfect personal portfolio website designed for creative professionals, UI/UX designers, and developers. Built with **HTML5, CSS3 & Bootstrap 5** — showcasing clean, semantic markup combined with powerful CSS techniques and Bootstrap's responsive grid system.

The website features a stunning hero section with a typewriter CSS animation, an interactive portfolio gallery with tab filtering and hover overlay effects, a timeline-based resume section, skill progress bars, client testimonials carousel, a contact form, and a fully responsive layout that adapts beautifully from desktop to mobile.

> 💡 **Why this project stands out:** It demonstrates advanced front-end development skills including CSS keyframe animations (typewriter effect), Bootstrap 5 tabs & carousel components, Flexbox layouts, responsive design with media queries, hover transitions, overlay effects, and a cohesive design system — all tied together with a warm orange accent theme.

## 🚀 Installation Instructions

### Prerequisites

- Any modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code recommended)
- Git (optional, for cloning)

### Steps

**1. Clone the repository:**

```bash
git clone https://github.com/waleedtarbosh/PortfolioBootstrap.git
```

**2. Navigate to the project directory:**

```bash
cd PortfolioBootstrap
```

**3. Open in your browser:**

```bash
# Simply open the index.html file in your browser
# Option 1: Double-click index.html
# Option 2: Right-click → Open with → Your browser
# Option 3: Use a live server (see below)
```

> ✅ **No dependencies to install!** No `npm install`, no build tools — Bootstrap & Font Awesome are loaded via CDN and local files.

---

## 💻 How to Run the Development Server

While you can open `index.html` directly, using a live server provides auto-reload on file changes:

### Option 1: VS Code Live Server (Recommended)

```
1. Install the "Live Server" extension in VS Code
2. Right-click on index.html
3. Select "Open with Live Server"
4. Browser opens at http://127.0.0.1:5500
```

### Option 2: Python HTTP Server

```bash
# Python 3
python -m http.server 8000

# Then open http://localhost:8000
```

### Option 3: Node.js HTTP Server

```bash
npx serve .

# Then open the URL shown in terminal
```

## 🎨 Design Decisions

### 🏗️ Architecture Choices

| Decision | Rationale |
|:---|:---|
| **Bootstrap 5 Framework** | Rapid responsive development with a powerful grid system and pre-built components |
| **Custom CSS on top** | Extends Bootstrap with unique animations, color scheme, and hover effects |
| **Single-page Layout** | Portfolio websites benefit from continuous scroll storytelling |
| **Tab-based Portfolio** | Allows users to filter projects by category without page reloads |

### 🎨 Visual Design Philosophy

| Decision | Rationale |
|:---|:---|
| **Orange & Dark Theme** (`#ff724c` + `#3b3b3b`) | Warm, energetic, and modern — conveys creativity and professionalism |
| **Dosis Headings** | Clean, geometric typeface with excellent readability at large sizes |
| **Roboto Body Text** | Google's versatile font designed for screens — optimal readability |
| **Typewriter Animation** | Hero section immediately grabs attention with dynamic text reveal |
| **Timeline Resume** | Visual storytelling of career progression with connected dot-line design |
| **Overlay Hover Effects** | Portfolio cards reveal search icon with smooth scaleY transitions |

### 📱 Responsive Strategy

```
Desktop (≥992px)     → Full layout, all animations, 4-column services, 3-column portfolio
Tablet (768-991px)   → Wrapped grids, collapsible nav, adjusted padding, centered resume
Mobile (≤575px)      → Single column, mobile hero background, adjusted typography
```
